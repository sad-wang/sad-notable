# 数据库文档
### t_user 表
| 字段名 | 类型 | 描述 |
|  ----  | ----  | ---- |
| user_id | String | 用户 ID |
| _pwd | String | 用户密码 |
| _record | Object | 用户录制的乐曲 |
### t_sheet_music 表
| 字段名 | 类型 | 描述 |
|  ----  | ----  | ---- |
| sheet_music_id | String | 乐谱 ID |
| _title | String | 乐谱标题 |
| _sheet | String | 乐谱 |
| _lyrics | String | 歌词 |
| _review | Arrary | 乐谱评论数组，由评论id构成 |
### t_review 表
| 字段名 | 类型 | 描述 |
|  ----  | ----  | ---- |
| review_id | String | 评论 ID |
| user_id | String | 用户 ID |
| sheet_music_id | String | 乐谱 ID |
| _content | String | 评论内容 |
