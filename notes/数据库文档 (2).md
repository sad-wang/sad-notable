# 数据库文档
### t_user 表
| 字段名 | 类型 | 描述 |
|  ----  | ----  | ---- |
| user_id | String | 用户 ID |
| user_pwd | String | 用户密码 |
| user_record | array | 用户录制的乐曲 |
### t_sheet_music 表
| 字段名 | 类型 | 描述 |
|  ----  | ----  | ---- |
| _id | String | 乐谱 ID |
| title | String | 乐谱标题 |
| sheet | String | 乐谱 |
| lyrics | String | 歌词 |
| review | Arrary | 乐谱评论数组，由评论构成 |
